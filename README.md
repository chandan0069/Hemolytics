🩸 HemoLytics – AI Powered Blood Disease Prediction System

HemoLytics is a Machine Learning based medical screening application that predicts blood-related diseases 
using laboratory blood test parameters and provides structured medical explanations, diet recommendations, and professional medical reports.

📌 Features

✔ Machine Learning based disease prediction
✔ Supports multiple blood parameters
✔ Doctor-style medical explanation
✔ Diet & precaution recommendations
✔ Hindi & English AI voice output
✔ Professional PDF medical report generation
✔ Interactive dashboard & analytics
✔ Modern responsive UI with animations

🧠 Workflow

User enters blood test values
ML model analyzes clinical patterns
Disease prediction is generated
Medical explanation & diet advice displayed
AI voice explanation (Hindi / English)
Downloadable medical PDF report generated
